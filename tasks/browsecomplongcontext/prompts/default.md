# Long-Context Retrieval Task

You are given a set of web pages and a question. The full context is in `context.txt` in your workspace.

## Instructions

1. Read `context.txt` — it contains multiple web pages followed by the question.
2. Find the answer using ONLY the provided documents. Do not browse the web.
3. Write your final answer to `answer.txt` as a concise phrase or name.

## Rules

- Base your answer strictly on the content in `context.txt`.
- Do not use web search or browser tools.
- `answer.txt` must contain only the answer value and nothing else (no "Final Answer:" prefix).
